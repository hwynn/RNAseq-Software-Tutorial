HTseq

Summary

counts are generated by finding the number of reads the overlap with a gene from our gene model. [Still Confused?]

Installation

For linux
  1. You must first install Pip before continuing.
  2. pip install HTSeq
[Installation Guide]

How to use this program

usage

htseq-count [options] <alignment_files> <gff_file>
[Read Further]

my command

htseq-count -s no -r pos -t exon -i Parent -f bam $RNApath/map/SRR1930186Aligned.sortedByCoord.out.bam \
$RNApath/genome/annotations/Cyanidioschyzon_merolae.ASM9120v1.39.chromosome.19.gff3 > $RNApath/counts/SRR1930186-output_basename.counts
