Bowtie2

Summary

Bowtie is a sequence alignment and mapping tool the works in conjunction with Tophat [Still Confused?]

Installation

For linux (specifically ubuntu)
  1. (optional but recommended)
    cd /home/your_username/
    mkdir toolkit if you haven't created such a folder already
    cd toolkit
  2. Get the installation file here or from the official site
    save the file as "bowtie2.zip"
    or use
    wget --output-document bowtie2.zip \
    https://sourceforge.net/projects/bowtie-bio/files/bowtie2/2.3.4.1/bowtie2-2.3.4.1-linux-x86_64.zip/download
  3. extract file
    unzip bowtie2.zip
  4. compile program and grant permissions
    cd /home/your_username/toolkit/bowtie2/bowtie2-2.3.3.1-linux-x86_64/
    make
    chmod 775 bowtie2
    chmod 775 bowtie2-build
    chmod 775 bowtie2-inspect

  5. Add the program to your PATH
    export PATH="$PATH:/home/your_username/toolkit/bowtie2/bowtie2-2.3.3.1-linux-x86_64/"
