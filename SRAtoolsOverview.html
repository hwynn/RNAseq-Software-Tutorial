SRAtoolkit

Summary

SRAtools has a function called fastq-dump. This will probably be the first step in your RNA-seq pipeline. This will download the SRA files you need to start your pipeline. [Still Confused?]

Installation

For linux (specifically ubuntu)
  1. (optional but recommended)
    cd /home/your_username/
    mkdir toolkit if you haven't created such a folder already
    cd toolkit
  2. Get the installation file here or from the official site
    save the file as "sratoolkit.tar.gz"
    or use
    wget --output-document sratoolkit.tar.gz http://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/current/sratoolkit.current-ubuntu64.tar.gz
  3. extract file
    tar -vxzf sratoolkit.tar.gz
  4. Add the program to your PATH
    export PATH="$PATH:/home/your_username/toolkit/sratoolkit/sratoolkit.2.9.1-1-ubuntu64/bin"
