Trimmomatic

Summary

Trimmomatic trims low quality reads off of raw sequence data before the mapping step. [Still Confused?]

Installation

For linux (specifically ubuntu)
  1. (optional but recommended)
    cd /home/your_username/
    mkdir toolkit if you haven't created such a folder already
    cd toolkit
  2. Get the installation file here or from the official site
    or use
    wget http://www.usadellab.org/cms/uploads/supplementary/Trimmomatic/Trimmomatic-0.38.zip
  3. extract file
    unzip Trimmomatic-0.38.zip
  4. In case java isn't already installed
    sudo apt install default-jre

usage

Single End Mode

java -jar <path to trimmomatic jar> SE [-threads <threads>] [-phred33 | -phred64]
[-trimlog <logFile>]
<input> <output> <step 1>


Paired End Mode

java -jar <path to trimmomatic.jar> PE [-threads <threads>] [-phred33 | -phred64]
[-trimlog <logFile>] >]
[-basein <inputBase> | <input 1> <input 2>]
[-baseout <outputBase> | <paired output 1> <unpaired output 1> <paired output 2> <unpaired output 2> <step 1>
