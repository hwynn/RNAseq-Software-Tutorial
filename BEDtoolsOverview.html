BEDtools

Summary

BEDtools is used for genome arithmetic. BEDtools intersect can be used to identify and count all regions in multiple files of genetic data overlap. However the results from BEDtools insersect will differ from those of htseq-count. [Still Confused?]

Installation

For linux (specifically ubuntu)
  1. (optional but recommended)
    cd /home/your_username/
    mkdir toolkit if you haven't created such a folder already
    cd toolkit
  2. Get the installation file here or from the official site
    save the file as "bedtools-2.27.1.tar.gz"
    or use
    wget --output-document bedtools-2.27.1.tar.gz https://github.com/arq5x/bedtools2/releases/download/v2.27.1/bedtools-2.27.1.tar.gz
  3. extract file
    tar xvfz bedtools-2.27.1.tar.gz
    cd bedtools2

  4. you may have to install zlib. Ask an admin for help if using a school computer.
    sudo apt-get install zlib1g-dev
    sudo apt-get install zlib

  5. compile
    make
  6. Add the program to your PATH
    export PATH="$PATH:/home/your_username/toolkit/bedtools2/bin"
