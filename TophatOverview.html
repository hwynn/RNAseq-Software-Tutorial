Tophat

Summary

Tophat indexes reads then maps them to a reference genome. [Still Confused?]

Installation

For linux (specifically ubuntu)
  1. (optional but recommended)
    cd /home/your_username/
    mkdir toolkit if you haven't created such a folder already
    cd toolkit
  2. You must first install Bowtie2 before continuing.
  3. Get the installation file here or from the official site
    save the file as "tophat-2.1.1.Linux_x86_64.tar"
    or use
    wget --output-document tophat-2.1.1.Linux_x86_64.tar \
    https://ccb.jhu.edu/software/tophat/downloads/tophat-2.1.1.Linux_x86_64.tar.gz

  4. extract file
    tar xvf tophat-2.1.1.Linux_x86_64.tar
  5. Add the program to your PATH
    export PATH="$PATH:/home/your_username/toolkit/tophat-2.1.1.Linux_x86_64"

How to use this program

Reminder: use
export PATH="$PATH:/home/your_username/toolkit/tophat-2.1.1.Linux_x86_64"

usage

tophat2 [options]* <genome_index_base> <reads1_1[,...,readsN_1]> [reads1_2,...readsN_2]
