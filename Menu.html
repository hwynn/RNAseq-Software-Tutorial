RNA-seq Software Guide

About this guide

The first section of this guide is devoted to reference pages for each of the software. They will explain how to install the software as well as a brief explanation as to how to use it. In each reference page, I show you what my commands looked like when running the software. This is is simply a visual demonstration of what a command might look like when you use the program. Don't actually try running those exact commands.


The second section of this guide will explain RNA-seq and provide a step by step walkthrough of a complete experiment using an RNA-seq pipeline. This guide provides links to several relevant sources and even other similar guides. Hopefully this reduces the amount of time you'll have to spend searching for answers. To reduce clutter, some of these links will be invisible until you hover over them. For the first section these will mostly be in the installation section and at the end of the typical usage section.
